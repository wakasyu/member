# トップページの写真

このフォルダに `1.jpg`, `2.jpg`, `3.jpg` ... という名前で写真を置くと、ログイン後の「トップ」タブの背景に自動で読み込まれて横流し（スライドショー）表示されます。

- ファイル名は連番の `.jpg`（最大 `12.jpg` まで）
- 存在しない番号は自動的に無視されるので、写真を足したり減らしたりするだけで反映されます（コードの変更は不要）
- 横長（16:9など）の写真が背景としてきれいに収まります。あまり大きすぎるファイルは表示が遅くなるので、長辺2000px程度・1枚1MB以下を目安に書き出してください
- 写真が1枚も無い場合はブランドカラーの単色背景になります
